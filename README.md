# AI Tank（Vue 红白机坦克大战）

基于 Vue 3 + Vite 的红白机风格坦克大战。包含单机与联机模式、道具系统、AI 坦克与基地防守玩法。

## 游戏功能

- 单机模式：守住基地，随机地图生成，基地外圈固定砖墙
- 联机模式：房间制双人对战，双方坦克可互相攻击
- NPC 坦克：自动追击玩家与基地，具备射击逻辑
- 道具体系：火力、速度、装甲、加固、爆破
- 场景元素：砖块、钢板、森林、水域
- 音效与背景音乐

## 游戏规则（单机）

- 胜负条件只有三种：
- 基地被摧毁即失败
- 玩家被击毁 3 次即失败
- 存活到 150 秒即胜利
- NPC 清空不会结束比赛，会持续刷新新一轮
- 基地位置固定，外圈砖块固定
- 地图随机生成但布局保持规律（模块化拼接）
- NPC 攻击目标玩家/基地比例为 1:1

## 游戏规则（联机）

- 双人对战，创建/加入房间后双方准备开始
- 地图随机生成，布局保持规律（模块化拼接）
- 双方可互相击杀，NPC 对双方无敌我之分
- 计分与胜负逻辑以游戏内 HUD 和提示为准

## 道具说明

- 火力：30 秒内两发击毁敌军
- 速度：30 秒内移动速度翻倍
- 装甲：本条命需要 6 发炮弹击毁
- 加固：基地周围 20 秒临时强化
- 爆破：清屏当前所有 NPC（不影响后续刷新）

## 操作方式

| 按键 | 功能 |
| --- | --- |
| 方向键 | 移动 |
| 空格 | 发射炮弹 |
| Enter | 失败/胜利后重开 |

移动端会出现触控按钮替代方向键与开火。

## 启动命令

安装依赖：

```bash
npm install
```

启动前端开发服务器：

```bash
npm run dev
```

启动联机服务（WebSocket）：

```bash
npm run server
```

构建生产包：

```bash
npm run build
```

本地预览构建产物：

```bash
npm run preview
```

## 联机说明

- 服务端默认端口：`10087`
- 前端可通过 URL 参数 `?ws=host:port` 指定服务端地址
- 也可修改 `src/App.vue` 内的 `WS_PORT`
