# AI Tank (Vue 坦克大战)

一个使用 Vue 3 + Vite 开发的经典坦克大战游戏。

## 🎮 游戏特性

- 经典坦克大战玩法
- 玩家控制坦克保卫基地
- 敌人 AI 自动攻击
- 多种地形元素（砖块、钢铁、森林、水域）
- 道具系统
- 分数统计

## 🚀 技术栈

- **框架**: Vue 3.5.13
- **构建工具**: Vite 6.2.4
- **语言**: JavaScript (ES Module)
- **渲染**: Canvas API

## 📦 安装

确保已安装 Node.js，然后运行：

```bash
npm install
```

## 🔧 开发

启动开发服务器：

```bash
npm run dev
```

## 🏗️ 构建

构建生产版本：

```bash
npm run build
```

## 👁️ 预览

预览生产构建：

```bash
npm run preview
```

## 🎯 游戏操作

| 按键 | 功能 |
|------|------|
| 方向键 ↑↓←→ | 移动坦克 |
| 空格键 | 发射子弹 |

## 🎨 游戏元素

### 地形
- 🧱 **砖块**: 可被摧毁（3 点生命值）
- ⬜ **钢铁**: 不可摧毁
- 🌲 **森林**: 提供掩护
- 💧 **水域**: 无法通过
- 🏠 **基地**: 需要保护的目标

### 角色
- **玩家坦克**: 黄色主体，橙色装饰
- **敌人坦克**: 灰色主体，红色装饰
- **子弹**: 白色弹道

## ⚙️ 游戏配置

主要游戏参数（可在 `src/App.vue` 中调整）：

- 地图大小：26×26 格
- 每格尺寸：16 像素
- 玩家速度：1.65
- 敌人速度：玩家的 17.5%
- 子弹速度：4.8
- 玩家射击冷却：260ms
- 最大玩家子弹数：1 发
